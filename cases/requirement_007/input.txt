## 1. 产品概述

### 1.1 产品简介

本组件提供原生鸿蒙聊天交互界面，支持完整的客服对话功能，包括文字、表情、图片、地址位置等基础交互。适用于需要集成客服聊天能力的鸿蒙应用。

### 1.2 产品形态

- **类型**: HarmonyOS HAR（HarmonyOS Ability Resource）组件包
- **版本**: 1.0.0
- **license**: Apache-2.0

## 2. 组件使用需求（分优先级）

### 2.1 P0 - 核心功能

| 序号 | 需求名称 | 需求描述 |
|------|----------|----------|----------|
| P0-1 | 文本消息发送 | 支持用户输入文本内容并发送，展示聊天消息气泡 |
| P0-2 | 文本消息接收 | 接收客服方文本消息并展示 | 
| P0-3 | 消息列表渲染 | 展示聊天消息列表，按时间顺序排列 | 
| P0-4 | 头像展示 | 支持客服/用户头像显示，可配置 | 
| P0-5 | 消息时间戳 | 显示消息发送时间，可配置显示策略 | 

### 2.2 P1 - 重要功能

| 序号 | 需求名称 | 需求描述 |
|------|----------|----------|----------|
| P1-1 | 图片发送 | 用户从相册选择图片发送 |
| P1-2 | 图片预览 | 点击查看大图，支持缩放 |
| P1-3 | 位置发送 | 选择地理位置并发送 |
| P1-4 | 位置查看 | 点击查看位置详情/地图 |
| P1-5 | 语音输入 | 长按说话转文字 |
| P1-6 | 拍照发送 | 调用相机拍照并发送 |

### 2.3 P2 - 增强功能1

| 序号 | 需求名称 | 需求描述 |
|------|----------|----------|----------|
| P2-1 | URL链接识别 | 自动识别消息中的URL并高亮显示 | 
| P2-2 | 链接跳转 | 点击URL可打开浏览器 |
| P2-3 | 历史消息加载 | 下拉刷新加载历史消息 | 
| P2-4 | 自定义功能入口 | 扩展输入框功能按钮 | 
| P2-5 | 头像点击回调 | 点击头像触发回调事件 | 
### 2.4 P3 - 增强功能2

| 序号 | 需求名称 | 需求描述 | 实现位置 |
|------|----------|----------|----------|
| P3-1 | 多设备适配 | 直板机/折叠屏/平板自适应布局 | ChatView.ets / GridRowColSetting |
| P3-2 | 键盘避让 | 软键盘弹起时自动调整布局 | ChatView.ets / WindowModel |
| P3-3 | 语音模式切换 | 文本输入/语音输入模式切换 | ChatInput.ets |

## 4. 操作步骤

### 4.1 消息发送操作

#### 4.1.1 发送文本消息

1. 用户在输入框中输入文本内容
2. 输入完成后，点击发送按钮（↑）或点击键盘回车键
3. 消息出现在聊天列表中
4. 客服端自动响应（Mock服务）

#### 4.1.2 发送图片消息

1. 点击输入框左侧 `+` 按钮
2. 在展开的功能面板中点击"图片"图标
3. 系统打开图库选择器
4. 选择图片（最多5张）后确���
5. 图片消息出现在聊天列表中

#### 4.1.3 发送拍照照片

1. 点击输入框左侧 `+` 按钮
2. 在展开的功能面板中点击"拍照"图标
3. 系统打开相机
4. 拍摄完成后确认
5. 照片消息出现在聊天列表中

#### 4.1.4 发送位置消息

1. 点击输入框左侧 `+` 按钮
2. 在展开的功能面板中点击"位置"图标
3. 系统打开地图位置选择器
4. 选择或搜索目标位置
5. 位置消息出现在聊天列表中

#### 4.1.5 发送语音转文字

1. 点击输入框左侧麦克风图标，切换到语音模式
2. 长按麦克风按钮并说话
3. 语音自动识别为文字并显示在输入框
4. 点击发送按钮发送

### 4.2 消息交互操作

#### 4.2.1 查看图片大图

1. 点击图片消息
2. 打开图片预览界面

#### 4.2.2 查看位置详情

1. 点击位置消息
2. 打开地图应用显示位置详情

#### 4.2.3 复制文本

1. 长按文本消息
2. 出现复制选项
3. 点击复制到剪贴板

#### 4.2.4 打开链接

1. 消息中有URL链接时，链接会高亮显示
2. 点击链接，自动打开浏览器

### 4.3 历史消息操作

1. 在消息列表顶部下拉
2. 触发刷新，加载历史消息
3. 历史消息添加到列表顶部

## 5. 约束规则

### 5.1 环境约束

| 约束项 | 要求 | 说明 |
|--------|------|------|
| DevEco Studio版本 | DevEco Studio 6.0.1 Release 及以上 | 集成开发环境 |
| HarmonyOS SDK版本 | HarmonyOS 6.0.1 Release SDK 及以上 | 鸿蒙SDK |
| 设备类型 | 华为手机（包括双折叠和阔折叠）、平板 | 仅支持华为设备 |
| 系统版本 | HarmonyOS 6.0.1(21) 及以上 | 系统版本要求 |

### 5.2 功能开关约束

| 开关名称 | 类型 | 默认值 | 说明 |
|----------|------|--------|------|
| showCamera | boolean | true | 是否显示拍照功能图标 |
| showPicture | boolean | true | 是否显示图片功能图标 |
| showLocation | boolean | true | 是否显示位置功能图标 |

### 5.43 数据约束

| 约束项 | 限制值 | 说明 |
|--------|--------|------|
| 图片选择数量 | 最多5张 | 单次选择图片数量限制 |
| 输入框提示文字 | 默认"请输入" | 可自定义配置 |
| 头像大小 | 默认40vp | 可通过avatarParam配置 |
| 头像圆角 | sys.float.corner_radius_level10 | 系统默认圆角 |

### 5.4 消息类型约束

| 消息类型 | 约束 | 说明 |
|----------|------|------|
| 文本消息 | 最大长度无限制 | 按实际输入 |
| 图片消息 | 支持格式jpg/png | 系统图库/相机 |
| 位置消息 | 需包含经纬度 | MapKit返回 |
| URL链接 | 自动识别http(s):// | 非http前缀不识别 |

### 5.5 设备适配约束

| 断点类型 | 屏幕宽度 | 布局特点 |
|----------|----------|----------|
| sm | < 600vp | 单列布局，功能项4个/页 |
| md | 600-840vp | 单列布局，功能项8个/页 |
| lg | 840-1200vp | 单列布局，功能项16个/页 |
| xl | >= 1200vp | 功能项16个/页 |

## 7. 数据模型

### 7.1 消息模型

```typescript
class MessageModel {
  id: string                    // 消息ID
  type: MessageType               // 消息类型(text/image/location/url)
  content: ResourceStr          // 消息内容
  timestamp: number             // 时间戳
  isShowTime: boolean           // 是否显示时间
  sender: SenderType           // 发送者类型(user/customer_service)
  status?: MessageStatus       // 发送状态(sending/sent/failed)
  imageAspectRatio?: number      // 图片宽高比
  locationInfo?: LocationInfo  // 位置信息
}
```

### 7.2 头像信息模型

```typescript
class AvatarInfo {
  avatar: ResourceStr           // 头像资源
  name: string                 // 昵称
}
```

### 7.3 头像参数模型

```typescript
interface AvatarParam {
  avatarSize?: number          // 头像大小，默认40
  avatarRadius?: Length        // 头像圆角
  avatarMargin?: number         // 头像间距
}
```

### 7.4 功能开关模型

```typescript
interface FeatureSwitch {
  showCamera?: boolean          // 是否显示拍照，默认true
  showPicture?: boolean        // 是否显示图片，默认true
  showLocation?: boolean       // 是否显示位置，默认true
}
```

### 7.5 自定义功能项模型

```typescript
interface FeatureItemConfig {
  icon: Resource               // 图标
  label: ResourceStr           // 标签(最多2字符)
  isSymbolGraph: boolean      // 是否为Symbol图标
  onClick: () => void      // 点击回调函数
}
```
